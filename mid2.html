Attacks & Exploits

Understanding Major Cybersecurity Threats & Real-World Exploits

DoS (Denial of Service)

  • Attack from a single source
  • Overloads server resources (CPU, memory, bandwidth)
  • Makes services unavailable to legitimate users
  • Example: Flooding a website with excessive requests

DDoS (Distributed DoS)

  • Attack from multiple compromised devices (botnet)
  • Harder to block due to many IP sources
  • Often uses infected IoT devices
  • Targets: Banks, gaming servers, e-commerce sites

SQL Injection (Concept)

  • Injection attack targeting database-driven applications
  • Occurs when unsanitized input is embedded into SQL queries
  • Allows attackers to bypass login systems
  • May expose, modify, or delete database records
  • Prevention: Prepared statements & input validation

XSS (Cross-Site Scripting) – Concept

  • Injection of malicious JavaScript into web pages
  • Executed in victim’s browser
  • Can steal session cookies or tokens
  • Types: Stored, Reflected, DOM-based
  • Prevention: Output encoding & Content Security Policy

IP Spoofing

  • Forging source IP address in network packets
  • Makes traffic appear from a trusted source
  • Used in DDoS amplification attacks
  • Difficult to trace attacker’s real location

Case Study: WannaCry (2017)

  • Global ransomware attack in May 2017
  • Exploited Windows SMB vulnerability (EternalBlue)
  • Spread automatically like a worm
  • Infected 200,000+ computers in 150+ countries

How WannaCry Worked

  • Scanned for vulnerable systems
  • Installed ransomware payload
  • Encrypted victim files
  • Demanded Bitcoin ransom payment
  • Caused billions in damages worldwide

Lessons Learned

  • Importance of regular patching & updates
  • Network segmentation limits malware spread
  • Maintain secure offline backups
  • Cybersecurity awareness training is critical
